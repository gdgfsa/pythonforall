Sobre o evento

Python For All é voltado para usuárixs/entusiastas da linguagem Python de todos os níveis de conhecimentos e com foco em atrair profissionais de outras áreas que procuram usar Python no seu dia-a-dia. O Python For All vem com uma proposta de ser um evento diverso e versátil da nossa tão valorosa comunidade, quer trazer todo tipo de gente para brincar/remixar/aprender/curtir Python em um ambiente com uma pegada cultural, com música ao vivo e oficina de forró. O For All remete tanto "para todos", um evento aberto a toda comunidade, quanto a referência de uma das lendas do surgimento do nome Forró.
